## 🆘 Troubleshooting Common Issues

**"My site isn't in Google's index"**
- Check Search Console for errors
- Verify sitemap is submitted
- Check robots.txt allows crawling
- Wait 2-4 weeks (new sites take time)

**"I have low CTR in search results"**
- Improve meta title (include primary keyword)
- Improve meta description (make it compelling)
- Add numbers/dates to titles
- Create urgency (limited offer, 2026, etc.)

**"My keywords aren't ranking"**
- Check competition level (use SEMrush)
- Wait 3-6 months (new content needs time)
- Check page is properly indexed
- Ensure keyword is in content naturally
- Build more backlinks

**"Traffic dropped suddenly"**
- Check Search Console for indexing issues
- Verify no core updates hit your niche
- Check competitors' activity
- Review on-page changes
- Check for manual penalty
